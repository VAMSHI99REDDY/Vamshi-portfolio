# Vamshi's Developer Portfolio 🚀

A high-performance, cinematic developer portfolio featuring a fully custom 3D web experience, a 60-FPS particle collision loading screen, and deep responsive design.

## Live Website 🌐

**👉 View Live:(https://vamshi-portfolio-nu.vercel.app/) 👈** *(Update this link to your actual Vercel deployment URL if different)*

## Features ✨

- **Cinematic Preloader:** A 4-second Apple-tier loading sequence built in HTML5 `<canvas>` with particle trails, collision physics, and lens flares.
- **3D Background:** Interactive cinematic dust and parallax effects using custom-rendered 3D image sequence frames.
- **Fully Responsive:** Sleek mobile navigation overlay, flawless grid systems, and typography scaling from 4K monitors down to 320px devices.
- **Dynamic Previews:** Automatic, crystal-clear website screenshots generated via the Microlink API to bypass iframe security restrictions.
- **Magnetic Buttons & Cursor:** Premium interactive hover states and a glowing ambient cursor.

## Tech Stack 🛠

- **HTML5** & **Vanilla JS** (Zero bloat, pure performance)
- **CSS3** (Custom properties, grid, clamp geometry, 60fps animations)
- **Canvas API** (Custom physics engine for preloader sparks)

## Local Development 💻

1. Clone the repository.
2. Start a local server:
   ```bash
   python -m http.server 8080
   ```
3. Open `http://localhost:8080` in your browser.
